# Type of Variable

# Int (Integer)

3, 4, 5, 100, 99, 0, -5

# Float/Double

3.5, 1.5, 1/2, 31.0

"HelloWorld"

# Boolean

True

False

  • Boolean can be only True or False.

# Naming Variable Dos and Don'ts

  • Start with a letter
  • Symbols and spaces are not allowed (-, /, #, @)
  • The underscore can be used (For example, first_name, last_name)
